Warpinator для Android — неофициальный порт одноимённого инструмента для обмена файлами из Linux Mint. Он полностью совместим с оригинальным протоколом и позволяет легко переносить файлы между устройствами Android и Linux.

Особенности:

- Автоматическое обнаружение совместимых сервисов в локальной сети
- Работает через Wi-Fi или точку доступа, подключение к Интернету не требуется
- Быстрая и простая передача файлов любого типа
- Приём целых каталогов
- Параллельная передача нескольких файлов
- Обмен файлами из других приложений
- Ограничение доступа с помощью группового кода
- Возможность запуска при загрузке системы
- Не требует определения вашего местоположения или каких-либо других необязательных разрешений
